Weather & Climate Analysis Dashboard developed using Power BI.

Project features:
- Imported and cleaned weather dataset (320+ rows)
- Created KPI cards for Average Temperature, Average Humidity, and Total Rainfall
- Built multiple visualizations including:
  - Temperature trend over time (Line Chart)
  - Rainfall distribution by city (Column Chart)
  - Weather condition analysis (Pie Chart)
  - Temperature vs Humidity relationship (Scatter Plot)
  - Wind speed analysis by city (Bar Chart)
  - Seasonal temperature comparison (Column Chart)
- Added interactive slicers for City, Season, Date, and Weather Condition
- Designed a user-friendly and visually appealing dashboard layout
- Included key insights for better data understanding

This project demonstrates skills in data visualization, dashboard design, and data analysis using Power BI.
